# Contributing to Simple Interest Calculator

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How Can I Contribute?

### Reporting Bugs
* Check the Issues tab to see if the bug has already been reported.
* If not, open a new issue, providing a clear description and steps to reproduce the bug.

### Suggesting Enhancements
* Open an issue to discuss your idea before implementing it.
* Describe the current behavior and the behavior you would like to see.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Commit your changes with clear messages.
4. Push to your fork and submit a pull request.

## Style Guidelines
* Maintain clean and readable code.
* Update the README.md if your changes introduce new functionality.
* Ensure all mathematical logic follows the standard simple interest formula.

Thank you for helping to improve this project!
