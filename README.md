# Simple Interest Calculator

A simple command-line tool that calculates simple interest based on the principal amount, annual interest rate, and time period in years. This project is part of a GitHub workflow exercise.

## Description
This application uses the mathematical formula $I = \frac{P \times R \times T}{100}$ to help users determine the interest earned or paid on a loan or investment. It is designed to be lightweight, accurate, and easy to use.

## Features
* Calculates Simple Interest and Total Amount.
* Supports decimal inputs for interest rates.
* Clean and user-friendly output.

## Usage
1. Provide the **Principal** amount.
2. Enter the **Annual Interest Rate** (as a percentage).
3. Enter the **Time Period** in years.
